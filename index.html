Step 0

사전 준비사항

Claude Code를 설치하기 전에 아래 항목들이 준비되어 있는지 확인하세요.

🟢

Node.js

Node.js 18 이상이 필요합니다. LTS 버전을 권장합니다.

≥ v18.x (권장 v22)
🔵

Anthropic API 키

console.anthropic.com에서 API 키를 발급받아야 합니다.

ANTHROPIC_API_KEY
🟣

GitHub 계정 + CLI

GitHub 계정과 gh CLI가 있으면 Git 워크플로우와 원활하게 연동됩니다.

gh auth login
설치 가이드

단계별 설치 방법

Windows PowerShell 기준으로 각 단계를 순서대로 따라하세요.

1

Node.js 설치 확인

PowerShell을 관리자 권한으로 실행하고 Node.js 버전을 확인합니다. 설치가 안 되어 있다면 winget 으로 바로 설치할 수 있습니다.

PowerShell (관리자)
PS> node --version # v18 이상이면 OK
PS> npm --version
 
# Node.js가 없다면 winget으로 설치
PS> winget install OpenJS.NodeJS.LTS
💡 설치 후 PowerShell을 새로 열어야 PATH가 적용됩니다. $env:PATH 변수 갱신이 필요할 수 있습니다.
2

Claude Code 글로벌 설치

npm을 통해 Claude Code를 전역으로 설치합니다. 설치 후 어디서든 claude 명령어를 사용할 수 있습니다.

PowerShell
PS> npm install -g @anthropic-ai/claude-code
 
# 설치 확인
PS> claude --version
✓ @anthropic-ai/claude-code@1.x.x
⚠️ PowerShell 스크립트 실행 정책 오류가 발생하면 Set-ExecutionPolicy RemoteSigned -Scope CurrentUser 를 먼저 실행하세요.
3

Anthropic API 키 설정

console.anthropic.com에서 API 키를 발급받은 후, 환경변수로 등록합니다. PowerShell 프로필에 추가하면 영구적으로 유지됩니다.

PowerShell — 환경변수 영구 등록
# 현재 세션에만 적용 (임시)
PS> $env:ANTHROPIC_API_KEY = "sk-ant-..."
 
# 영구 적용 — 사용자 환경변수로 저장
PS> [System.Environment]::SetEnvironmentVariable(
  "ANTHROPIC_API_KEY",
  "sk-ant-...",
  "User"
)
ℹ️ 환경변수 등록 후 PowerShell을 재시작해야 적용됩니다. claude 명령 실행 시 자동으로 키를 읽어옵니다.
4

Claude Code 첫 실행 및 인증

프로젝트 폴더로 이동한 뒤 claude 명령을 입력합니다. 최초 실행 시 이용약관 동의 절차가 진행됩니다.

PowerShell
# 프로젝트 폴더로 이동
PS> cd C:\Users\neurophet\Documents\myproject
 
# Claude Code 시작
PS> claude
✓ Claude Code 시작됨
  ╔══════════════════════════════╗
  ║ Welcome to Claude Code! ║
  ╚══════════════════════════════╝
5

GitHub 연동

GitHub CLI(gh)를 설치하고 인증하면 Claude Code에서 PR 생성, 이슈 확인, 코드 푸시 등을 자연어로 처리할 수 있습니다.

PowerShell — GitHub CLI 설치 및 인증
# GitHub CLI 설치
PS> winget install GitHub.cli
 
# GitHub 로그인 (Edge 브라우저로 OAuth 인증)
PS> gh auth login
? What account do you want to log into?
> GitHub.com
? Preferred protocol: HTTPS
? Authenticate Git with your GitHub credentials? Yes
✓ Logged in as DanielHan-Code

내 GitHub 레포지토리에서 바로 시작하기:

revapp

기존 프로젝트에서 Claude Code로 클론 후 개발을 시작합니다.

PS> gh repo clone DanielHan-Code/revapp
PS> cd revapp && claude
→ github.com/DanielHan-Code/revapp

hrstudy

학습 프로젝트에서 Claude Code와 함께 코드 리뷰 및 기능 추가를 진행합니다.

PS> gh repo clone DanielHan-Code/hrstudy
PS> cd hrstudy && claude
→ github.com/DanielHan-Code/hrstudy
6

자주 쓰는 Claude Code 명령어

Claude Code에서 자주 활용하는 슬래시 커맨드와 작업 패턴입니다.

Claude Code — 슬래시 명령어
# GitHub PR 생성
> /commit — 변경사항 커밋 자동화
 
# 코드 리뷰 요청
> 이 코드의 버그를 찾아서 수정해줘
 
# 새 기능 구현
> 로그인 기능에 소셜 로그인을 추가해줘
 
# 테스트 실행
> npm test 실행하고 실패한 테스트 고쳐줘
 
# 도움말
> /help — 사용 가능한 모든 명령어 보기
