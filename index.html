Claude Code란?
AI 코딩 도우미를
터미널에서 사용한다

채팅창에 말하듯 요청하면 AI가 실제 코드를 작성·수정·실행해 줍니다.

💬

대화하듯 개발

"로그인 기능 만들어줘", "이 버그 고쳐줘"처럼 말하면 Claude가 직접 코드를 작성하고 파일을 수정합니다.

⚡

터미널에서 바로

별도 앱 없이 PowerShell에서 claude 명령 하나로 실행됩니다. VS Code와도 연동됩니다.

🔌

MCP로 무한 확장

playwright, firecrawl 등 MCP 서버를 연결하면 웹 브라우저 자동화, 스크래핑 등 고급 기능을 사용할 수 있습니다.

완성 후엔
이런 게 가능해집니다

튜터리얼을 마치면 이런 작업들을 AI에게 말로 시킬 수 있습니다.

코드 작성

"이 기능 만들어줘"

요구사항을 자연어로 설명하면 실제 코드 파일을 생성·수정해 줍니다.

버그 수정

"이 에러 고쳐줘"

에러 메시지를 붙여넣으면 원인을 분석하고 코드를 자동으로 수정합니다.

웹 자동화

"이 사이트에서 데이터 가져와"

playwright MCP로 실제 브라우저를 조작해 웹 스크래핑·자동화를 수행합니다.

GitHub

"변경사항 커밋하고 PR 만들어줘"

코드 변경부터 커밋 메시지 작성, PR 생성까지 한 번에 처리합니다.

웹 검색

"최신 자료 찾아서 요약해줘"

firecrawl MCP로 웹페이지를 읽고 필요한 정보를 추출·정리합니다.

복잡한 작업

"이 작업 단계별로 계획 세워줘"

sequential-thinking MCP로 복잡한 문제를 체계적으로 분석·해결합니다.
