💬 배포 프롬프트
방금 만든 index.html을 GitHub Pages로 배포해줘: 1. gh repo create sales-dashboard --public 으로 레포 생성 2. git init, 파일 추가, 첫 커밋, push 3. GitHub Pages 활성화 방법 알려줘 (Settings → Pages → main 브랜치) 4. 최종 배포 URL 알려줘
💡
배포 URL은 https://내아이디.github.io/sales-dashboard 형태입니다. Push 후 1~2분 뒤 접속 가능합니다.
