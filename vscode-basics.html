Step 1

폴더 열기

Claude Code는 폴더(프로젝트) 단위로 작업합니다.
작업할 폴더를 VS Code에 먼저 열어야 Claude와 함께 그 안의 파일을 다룰 수 있습니다.

왜 폴더를 열어야 하나요?

ℹ️
Claude Code는 열려 있는 폴더 안의 파일들을 읽고 수정합니다.
폴더를 열지 않으면 Claude가 "어디에 파일을 만들까요?" 할 때 저장 위치가 없어 곤란해집니다.
작업 시작 전에 항상 폴더부터 여세요.

폴더 여는 방법 — 3가지

1

메뉴에서 열기 (가장 기본)

VS Code 상단 메뉴 File → Open Folder… 클릭 → 폴더 선택 → 폴더 선택 버튼

2

드래그 앤 드롭 (제일 간편)

윈도우 파일 탐색기에서 폴더 아이콘을 마우스로 잡아 VS Code 창 위에 놓으면 바로 열립니다.

3

단축키

Ctrl+K 를 눌렀다 떼고, 바로 Ctrl+O

Step 2

탐색기 (Explorer)

왼쪽 맨 위 📁 아이콘을 누르면 열리는 패널입니다.
프로젝트 안의 파일과 폴더 목록을 보여주는 곳이에요.

탐색기로 할 수 있는 것들

👁

파일 열기

파일 이름을 클릭하면 오른쪽 에디터 영역에서 내용을 볼 수 있습니다.

📄

새 파일 / 새 폴더 만들기

탐색기 패널에 마우스를 올리면 상단에 📄+ (새 파일), 📁+ (새 폴더) 아이콘이 나타납니다.
클릭 → 이름 입력 → Enter

🖱

파일 우클릭 → 이름 바꾸기 / 삭제

파일 위에서 마우스 오른쪽 버튼을 클릭하면 이름 변경(Rename), 삭제(Delete) 등 메뉴가 나타납니다.

Claude Code와 함께 쓸 때

💡
Claude가 파일을 새로 만들거나 수정하면 탐색기에 바로 반영됩니다.
파일이 잘 만들어졌는지 탐색기를 보면서 확인하세요.
수정된 파일 옆에는 작은 점(●)이 표시되어 저장이 안 된 상태를 알려줍니다.

유용한 단축키

기능단축키
탐색기 열기/닫기 Ctrl+Shift+E
파일 이름 변경 F2
파일 이름으로 빠르게 열기 Ctrl+P 후 파일명 입력
ℹ️
Ctrl+P 는 파일명 일부만 입력해도 찾아줍니다. 파일이 많아질수록 정말 편리합니다.

파일 위에서 우클릭해 메뉴를 확인해봤다

Step 3

소스 컨트롤

왼쪽 🔀 아이콘을 누르면 열리는 패널입니다.
Claude가 파일을 수정했을 때 어떤 내용이 바뀌었는지 확인하는 곳입니다.

이 패널이 왜 중요한가요?

ℹ️
Claude Code에게 "이 파일 수정해줘"라고 부탁하면, Claude가 파일을 바꿉니다.
소스 컨트롤 패널에서 정확히 어떤 줄이 추가되고 삭제됐는지 빨간색/초록색으로 확인할 수 있어요.
"Claude가 뭘 바꿨지?" 하는 궁금증이 생길 때 여기를 보세요.

M = 수정된 파일 (Modified)
U = 새로 생긴 파일 (Untracked)
D = 삭제된 파일 (Deleted)

파일 이름을 클릭하면 오른쪽에 빨간색(삭제된 줄)과 초록색(추가된 줄)으로 변경 내용을 한눈에 볼 수 있습니다.

변경 내용이 마음에 들 때 — 저장(커밋)하기

1

변경된 파일 옆 + 클릭

파일 위에 마우스를 올리면 + 가 나옵니다. 저장할 파일을 + 로 선택합니다.
전체 선택은 "변경 사항" 제목 옆 + 클릭

2

메시지 입력칸에 간단히 내용 적기

예: Claude가 메인 페이지 수정 — 나중에 어떤 변경이었는지 알아보기 위한 메모입니다.

✓

✓ 커밋 버튼 클릭

이 시점의 파일 상태가 기록됩니다. 나중에 뭔가 잘못되면 이 시점으로 되돌릴 수 있습니다.

⚠️
커밋을 꼭 해야 하는 건 아닙니다. 하지만 Claude와 함께 큰 변경을 하기 전에 커밋해 두면 "실수해도 돌아갈 수 있다"는 안전망이 생깁니다.
