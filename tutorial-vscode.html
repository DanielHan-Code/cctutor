Step 1

VS Code 설치

VS Code는 Microsoft가 만든 무료 코드 편집기입니다.
메모장보다 훨씬 강력하고, 전 세계 개발자들이 가장 많이 쓰는 도구예요.

🖱️ GUI — 클릭만으로 설치

VS Code가 뭔가요?

✏️
코드 편집기
메모장처럼 글자를 쓰고 수정할 수 있는데, 코드를 위해 특별히 만들어진 프로그램입니다
🆓
완전 무료
Microsoft가 만들었지만 누구나 무료로 사용할 수 있습니다. 설치도 간단합니다
🔌
익스텐션 지원
플러그인처럼 기능을 추가할 수 있어요. Claude Code도 이 방식으로 설치합니다
ℹ️ 이미 VS Code가 설치되어 있다면 이 단계를 건너뛰고 바로 다음 → 버튼을 눌러도 됩니다.

① VS Code 다운로드

⬇️ code.visualstudio.com/download 공식 다운로드 페이지 ↗
1

위 링크를 클릭해 다운로드 페이지로 이동

파란색 Windows 버튼을 클릭하면 설치 파일(.exe)이 다운로드됩니다.

2

다운로드된 파일 실행

브라우저 하단 또는 다운로드 폴더에서 VSCodeSetup-x64-…exe 파일을 더블클릭합니다.

3

설치 마법사 진행

약관 동의 → 다음 계속 클릭 → 설치 클릭. 특별히 바꿀 옵션 없이 기본값으로 진행해도 됩니다.

✓

설치 완료 → VS Code 실행

설치 마지막 화면에서 "Visual Studio Code 시작" 체크박스가 선택된 채로 마침을 클릭하면 바로 실행됩니다.

Step 2

VS Code 익스텐션 설치

VS Code 마켓플레이스에서 Claude Code 익스텐션을 설치합니다.
별도의 Node.js 설치나 터미널 작업 없이 클릭 몇 번으로 완료됩니다.

🖱️ GUI — 클릭만으로 설치
💡
PowerShell로 Claude Code CLI를 이미 설치한 분이라면 익스텐션을 추가해도 충돌 없이 함께 사용할 수 있습니다.

2

검색창에 Claude Code 입력

아래와 같이 Anthropic의 Claude Code 익스텐션이 목록에 나타납니다.

Step 3

Claude 계정 연동

사이드바의 Claude 아이콘을 클릭해 로그인합니다.
사용 중인 요금제에 따라 방식을 선택하세요.

🖱️ GUI — 클릭만으로 설치
✅

이미 로그인된 상태라면 이 단계를 건너뛰세요

왼쪽 사이드바에 이전 대화기록이 보인다면 이미 인증이 완료된 것입니다. 바로 Step 4 기본 사용법으로 넘어가세요.

① Claude Code 패널 열기

1

왼쪽 Activity Bar에서 Claude 아이콘 클릭

Claude Code 사이드 패널이 열리고 로그인 화면이 나타납니다.

2

또는 명령 팔레트에서 실행

Ctrl + Shift + P → Claude: Sign In 입력 후 Enter

② 연결 방식 선택

③ 로그인 진행

1

Sign in with Claude.ai 버튼 클릭

브라우저가 자동으로 열리며 claude.ai 로그인 페이지로 이동합니다.

2

Pro / Max 구독 계정으로 로그인

이메일·비밀번호 또는 소셜 로그인으로 인증합니다.

✓

VS Code로 돌아오면 자동 연동 완료

Claude Code 패널에 채팅창이 열립니다. 바로 사용 가능합니다.

✅ 완료 확인

브라우저에서 claude.ai 로그인 완료
VS Code Claude 패널에 채팅창이 나타났다

③ API 키 입력

1

Use API Key 선택

2

API 키가 없다면 먼저 발급

✓

키 입력 후 Enter → 연동 완료

① Claude 채팅 패널

열기

또는 Ctrl+Shift+P → Claude: Open Chat
