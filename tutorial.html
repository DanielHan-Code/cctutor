Step 1

Node.js + Git 설치

Claude Code가 동작하려면 Node.js와 Git이 반드시 먼저 설치되어 있어야 합니다.
이 단계만 PowerShell 명령어를 직접 실행합니다.

⌨️ 직접 명령어 입력

PowerShell을 관리자 권한으로 실행하세요

시작 메뉴에서 PowerShell 검색 → 우클릭 → 관리자 권한으로 실행

① Node.js 설치

⬇️ nodejs.org/download Windows 설치 파일 ↗
PowerShell (관리자)
PS> winget install OpenJS.NodeJS.LTS
설치 완료 후 PowerShell을 닫고 다시 열기
PowerShell — 설치 확인
PS> node --version # v18 이상이면 OK

② Git 설치

⬇️ git-scm.com/download/win Windows 설치 파일 ↗
PowerShell (관리자)
PS> winget install Git.Git
설치 완료 후 PowerShell을 닫고 다시 열기
PowerShell — 설치 확인
PS> git --version # 버전이 출력되면 OK
ℹ️ 두 가지 모두 설치 후 반드시 PowerShell을 완전히 닫고 다시 열어야 명령어가 인식됩니다.

✅ 완료 확인

node --version 실행 시 v18 이상 출력됨
git --version 실행 시 버전 출력됨
Step 2

Claude Code 설치

npm 한 줄로 설치합니다. 이후로는 명령어를 직접 입력할 일이 거의 없습니다.

⌨️ 직접 명령어 입력

① 글로벌 설치

📄 docs.anthropic.com/claude-code 공식 문서 ↗
PowerShell
PS> npm install -g @anthropic-ai/claude-code
⚠️ 오류가 나면 먼저 실행: Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

② 설치 확인

PowerShell
PS> claude --version
@anthropic-ai/claude-code@1.x.x

③ 최초 실행 — 초기 설정

PowerShell — Claude Code 실행
PS> claude

최초 실행 시 아래 순서로 초기 설정 화면이 나옵니다. 순서대로 따라하세요.

2

컬러 테마 선택

다크 / 라이트 / 다크-달록 등 테마를 고릅니다. 방향키로 선택 후 Enter.

💡 나중에 바꾸려면 Claude Code 대화창에서 /config 입력 후 변경 가능합니다.

자동 업데이트 설정

자동 업데이트 여부를 묻습니다. 권장: Y (항상 최신 버전 유지).

Step 5

GitHub 연동

GitHub 계정이 없어도 괜찮습니다. Claude가 pwr로 가입부터 gh CLI 설치·인증까지 안내합니다.

✦ 프롬프트 붙여넣기

① Claude Code 대화창에 붙여넣기

🌐 github.com 공식 사이트 ↗ ⬇️ cli.github.com Windows 설치 파일 ↗
📋 Claude Code 대화창에 붙여넣기
GitHub 연동을 도와줘. 아래 순서로 진행해줘: 1. winget으로 GitHub CLI(gh) 설치 winget install GitHub.cli 2. pwr을 사용해서 github.com 에 접속해줘 - 계정이 없으면 회원가입 페이지로 이동해줘 - 계정이 있으면 로그인 페이지로 이동해줘 - 내가 직접 가입/로그인할게. 완료되면 알려줄게 3. 내가 "완료했어"라고 하면 PowerShell에서 GitHub CLI 인증을 진행해줘: gh auth login (HTTPS 방식, 브라우저 인증 선택) 4. 인증 완료 후 gh auth status 로 로그인 확인해줘
⚠️ GitHub 가입/로그인은 직접 입력해야 합니다. 완료 후 "완료했어" 라고 입력하면 Claude가 인증을 이어서 진행합니다.

② 예상 흐름

1

Claude가 winget으로 gh CLI 설치

winget install GitHub.cli 자동 실행

2

pwr로 github.com 접속

Edge 브라우저로 가입/로그인 페이지 자동 이동

3

직접 가입/로그인 → "완료했어" 입력

이메일·비밀번호 직접 입력 후 Claude에게 알리기

✓

gh auth login 자동 진행 → 인증 완료

gh auth status에서 로그인 확인

③ 나만의 첫 번째 레포지토리 만들기

gh CLI로 GitHub 레포를 만들고, 로컬 폴더와 연결해 첫 커밋까지 올려봅니다.

📋 새 레포 생성 프롬프트 (붙여넣기)
내 첫 번째 GitHub 레포지토리를 만들어줘. 아래 순서대로 진행해줘: 1. gh repo create my-first-project --public --description "나의 첫 번째 프로젝트" 로 레포 생성 2. 로컬에 폴더 만들고 git 초기화: mkdir my-first-project cd my-first-project git init 3. README.md 파일 만들어줘 (프로젝트 이름과 간단한 설명 포함) 4. 첫 커밋하고 GitHub에 푸시해줘: git add . git commit -m "first commit" git branch -M main git remote add origin (레포 URL) git push -u origin main 5. 완료 후 gh repo view --web 으로 브라우저에서 열어줘
